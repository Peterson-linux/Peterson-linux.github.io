Red Hat Enterprise Linux 6.5

Cacti官网

  • [Cacti] : http://www.cacti.net/
    Cacti是一套基于PHP,MySQL,SNMP(数据采集)及RRDTool(数据存储和图表的生成)开发的网络流量监测图形分析工具。

    实验环境

监控机server16

config.php

访问：http://172.25.0.16/cacti/


刷新网页

默认登陆用户名和密码都为admin



被监控的远程主机server17

snmpd.conf




在监控机server16上检测一下

通过浏览器访问http://172.25.0.16/cacti/进行测试















Cacti的功能其实非常多，若要进一步去了解，可以自己查找相关说明书，这里只做一个初步的认识

Cacti开启spine

监控机server16

spine.conf

在浏览器中访问http://172.25.0.16/cacti/，settings中的Poller中的poller type改为spine,在Paths中的Spine Poller File Path 写上路径/usr/local/spine/bin/spine，设置好后save。


Cacti添加插件monitor

在浏览器访问ip/cacti–>Plugin Management–>安装–>开启–>点击monitor使用
